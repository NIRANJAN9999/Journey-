How I got here

  1. Where it started

    Arduino, HC-05 Bluetooth modules, relay-based home automation. Learned mostly by breaking things and working backward to figure out why.

  2. Going deeper

    ESP32 BLE devices, RGB status systems, an STM32 dashboard simulated end to end. Started reading datasheets for fun, not just when something broke.

  3. Systems, not just circuits

    Set up a self-hosted server from scratch and built a multi-agent system on top of it. Some days the soldering iron stays off and it's all SSH.
